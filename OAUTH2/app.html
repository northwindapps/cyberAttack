OAuth2 Authorization Code Flow

Step 1 — App redirects user to Auth Server with client_id + scopes
Step 2 — User logs in and approves on Auth Server
Step 3 — Auth Server redirects back with a one-time auth code
Step 4 — App exchanges auth code for an access token (server-to-server)
Step 5 — App uses token to fetch user data
